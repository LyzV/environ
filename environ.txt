Горячие клавиши терминала:
ctrl-d - ввод закончен. Выход из терминала.
ctrl-h - backspace
ctrl-c - останов программы
ctrl-u - удалить набранную строку
ctrl-s/ctrl-q - приостанов/продолжение вывода на терминал

Список полезных команд
who - кто в системе? Эквивалент - whoami
wc - word counter
	$ wc файл
grep - команда для обработки по шаблону. 
	$ grep шаблон файл
sort - сортировка строк из файла
	$ sort файл
tail - печать только части файла
	$ tail -1 poem  - печать последней строки
	$ tail +3 poem  - печать начиная с третьей строки
cmp - сравнивает два файла. Говорит только о том, что файлы не равны. Применяется и для бинарных файлов.
	$ cmp poem poem2
diff - сравнивает два файла. Печатает все различия. Обрабатывает только текстовые файлы.
	$ diff poem poem2
	Вот так можно сделать патч к коду:
	$ diff -u poem2 poem > patch_of_poem      Здесь poem2 - это оригинальный файл. Опция u - универсальный формат.
	А вот так можно применить патч к оригиналу (poem2)
	$ patch poem2 < patch_of_poem	Теперь poem2 стал таким-же как и poem.
rmdir - команда удаления директории!
